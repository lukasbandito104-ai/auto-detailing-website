# Deployment Instructions

## Deploying on Netlify

1. Go to [Netlify](https://www.netlify.com/).
2. Sign up or log in to your account.
3. Click on 'New Site from Git'.
4. Connect your GitHub repository and select the `auto-detailing-website` repo.
5. Configure the branch (typically `main`) and the build command (if applicable).
6. Set the publish directory to the appropriate folder (e.g., `build` or `dist`).
7. Click 'Deploy Site'.
8. Once deployed, you can manage your site settings in the Netlify dashboard.

## Deploying on Vercel

1. Go to [Vercel](https://vercel.com/).
2. Sign up or log in to your account.
3. Click on 'New Project'.
4. Import your GitHub repository and select the `auto-detailing-website` repo.
5. Configure the project settings, such as the framework and build settings.
6. Click 'Deploy'.
7. Your site will be live after the build process, and you can view it in your Vercel dashboard.
8. Customize your domain settings if needed.
